Public Roadmap

What I’m building next for the HBCU Research Repository

Last updated:

Now

Active

Shipping + validating with users.

  • Collecting feedback from researchers, students, faculty, and librarians
  • Improving search + filters based on real usage
  • Fixing metadata issues that hurt discoverability
  • Making citations easier to copy and reuse

Next

Planned

High-confidence improvements (no dates).

  • Adding datasets, books, and other academic content
  • Saved searches / bookmarks
  • Cleaner topic organization + normalization
  • Citation export formats (BibTeX / RIS)
  • Better browsing by institution
  • Personalized reccomendations to email

Later

Direction

Longer-term direction as the dataset grows.

  • Personalized discovery + alerts
  • Author and institution profiles
  • Deeper citation / impact insights
  • Bulk access or API options for researchers & libraries
  • Include more data sources such as arxiv, PubMed, and Google Scholar
